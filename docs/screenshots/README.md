# 스크린샷 폴더

README 본문에서 참조할 **3~5장**의 PNG(또는 WebP)를 이 디렉터리에 저장하세요.

권장 파일명:

1. `01-map-overview.png` — 지도 + 필터/사이드바
2. `02-charts.png` — 추이·막대 차트
3. `03-calendar.png` — 캘린더 페이지(있다면)
4. `04-export.png` — CSV 또는 표 영역
5. `05-mobile.png` — 좁은 화면에서 사이드바·지도 배치(선택)

Git에 올리기 싫다면 `.gitignore`에 `docs/screenshots/*.png` 를 추가하고, 제출 ZIP에만 포함해도 됩니다.
